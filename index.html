To create a table in HTML, you can use the <table> tag. Other tags include:

  • <th> Defines a header cell
  • <tr> Defines a row
  • <td> Defines a cell
  • <caption> Defines a table caption
  • <colgroup> Specifies a group of columns for formatting

You can also add attributes to modify the appearance and behavior of a table. For example, the rowspan attribute allows you to merge rows by extending them into multiple rows and the colspan attribute allows you to merge columns by extending them into multiple columns.
